User guide
==========

Overview:
---------
ipyreactwidget is a complementary project to ipywidget aiming to provide widgets with visual built using React framework. 
The main goal is to provide UI components from Material UI, which is a library of UI components implementing UI standards, 
Allowing its user to create great looking, easy to use applications. We specifically aim to provide a Material UI implementation 
to the jupyter world, in order to ease dashboard building process on a Jupyter Notebook.
